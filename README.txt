Bienvenue dans le site Camp Job finder.

Le but du site est de permettre à des animateurs de se trouver un camp où travailler, et à des camps de se trouver du personnel. 
Dans cette itération du projet, et vu le temps disponible, seul la création d'un compte "animateur" est possible.
Le site devait aussi permettre à un-e administrateur de se connecter et de vérifier la liste des postulations.

--> Pour commencer, rendez vous sur la page d'accueil. Pour ce faire, vous n'avez qu`à accédez au dossier /Web/ du projet avec un navigateur.
	
	--> Grâce à Bootstrap 4, le site est pleinement responsive. 
	--> Sur la page d'accueil, deux boutons sont disponibles: créer un compte et se connecter
	--> Cliquez sur "créer un compte"
	--> Ensuite, entrez des informations relativement valides, vous pouvez inventer l'adresse courriel 
	-- mais sâchez qu'un courriel de confirmation est normalement envoyé lors de la création d'un compte.
	-- j'en ferai la démonstration. 
	
	--> Une fois le compte créé et une fois retourné sur la page principale, 
	--> il est maintenant temps de se connecter. 
	--> Entrez le courriel utilisé pour la création de compte ainsi que le mot de passe choisis lors de l'inscription 
	--> Vous êtes maintenant connecté en tant qu'animateur!

	==> Vous pouvez aussi vous connecter avec l'animateur adf@test.com, 1234
	
	--> Sur la page de recherche d'un animateur plusieurs options s'offrent.
	
	--> Vous pouvez bien sûr rechercher une offre d'emploi avec des critères spécifiques.
	
	--> Vous pouvez aussi uploader une photo de profil ou écrire une courte présentation. 
	
	--> En haut, deux liens sont disponibles: l'un mène à la page actuelle alors que l'autre mène à 
	-- la page listant toutes les postulations effectuées affichant le nom du camp et la date.

	--> Vous pouvez maitenant vous déconnecter. 

	--> Dans cette itération, il est aussi possible de se connecter en tant qu'administrateur.

	--> L'utilisateur felix@test.com avec le mot de passe 1234 vous connectera en tant qu'administrateur.

	--> L'admin a seulement accès à la liste de toutes les postulations effectuées sur le site 
	
	--> avec le courriel de l'utilisateur, le camp en question, et la date de postulation.
